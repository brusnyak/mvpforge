# MVPForge.ai

MVPForge.ai is a tool that generates validated startup ideas and complete one-file MVPs using AI.

## Features

- **Idea Generator**: Deep, niche AI ideas tailored for indie hackers
- **Idea Validator**: Instant deep research reports for any startup idea
- **MVP Builder**: Generate complete one-file HTML/JS/Tailwind apps with AI integration
- **Founder Dashboard**: Track your ideas and get AI-powered feedback

## Setup

1. Get an OpenRouter API key (free tier available)
2. Add your API key to the app
3. Start generating ideas and MVPs!

## Deployment

This project is ready for deployment to Vercel:

1. Connect your GitHub repository to Vercel
2. Set environment variables if needed
3. Deploy!
